- **[lyra-qa-design](./lyra-qa-design/SKILL.md)** — Visual QA agent — audits live web apps with a designer's eye.
- **[lyra-website-planning](./lyra-website-planning/SKILL.md)** — Architect web platforms with PRDs, customer journey maps,
  and user personas.
